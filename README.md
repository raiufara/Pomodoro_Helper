# MinimaPomoTimer

Simple Focus Timer with personal and business work modes.

Version: `v1.1.0`

## Overview

MinimaPomoTimer is a lightweight web app for focus sessions and meeting timing.
It runs as a single-page app and keeps the interface simple while offering two visual modes:

- Personal mode for a warmer pomodoro-style focus experience
- Business mode for a calmer meeting and task-management view
- Light and dark appearance switching
- Local browser storage for mode preference and ToDo items

## What Changed in v1.1.0

- Renamed the app to `MinimaPomoTimer - Simple Focus Timer`
- Refreshed the UI theme for both personal and business modes
- Added dark mode support
- Improved responsive behavior in the header for smaller window widths
- Reworked the app icon, header logo, and favicon
- Tuned theme-aware browser color handling for a more polished tab experience

## Files

- `index.html`: main application UI and logic
- `tomato-icon.svg`: full app icon used in the header and touch icon
- `favicon.svg`: simplified favicon optimized for browser tabs

## Usage

Open `index.html` in a browser, or use the published GitHub Pages site if configured for this repository.
